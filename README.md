## Swachh Vision: Garbage Detection System

AI-powered system to detect garbage in real-time using YOLOv8 and OpenCV, with automated alerts for smart waste management.

## Features
- Real-time garbage detection  
- Works on images & live video  
- Automated alert system  



## Tech Stack
Python, YOLOv8, OpenCV, NumPy  



## Usage
bash
git clone https://github.com/Chaitanya-Tambe18/swachh-vision-garbage-detection.git
cd swachh-vision-garbage-detection
pip install -r requirements.txt
python main.py
